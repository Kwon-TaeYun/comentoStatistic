# 1. 2차 API 문서 (V2.0)

## 📌 1. 접속자 수 조회

### 🔹 [GET] /comento/users - 접속자 수 조회

설명  
- 서비스에 접속한 사용자 수를 조회하는 API

요청 정보  
- Method: GET  
- URL: http://localhost/comento/users  
- Headers: 없음  
- Body: 없음  

응답 예시 (200 OK)  
{
  "connectedUserCount": 128
}

에러 응답  
- 500 INTERNAL_SERVER_ERROR : 서버 오류
